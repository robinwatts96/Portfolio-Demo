About me

Learning to code has been a difficult journey, however, I have never done something so rewarding. 18 months ago I couldn't tell you what a div was, now I'm able to use my HTML and CSS skills to create responsive, custom sites which has been a great learning experience. More recently, I've enjoyed spending time implementing JavaScript (including API's and frameworks) to add interactivity to my projects.

I love to learn - throughout my life my drive has always been music, picking up an instrument and dedicating time and passion to learning to play a bit better than the day before. Though I still love music, over the past few years my free time has been dedicated to learning front-end development. I love it. Seeing or creating a design, building it, making it work how I want it to work. This is something I want to do every day.
